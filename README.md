# PhishGuard AI

PhishGuard AI is a real-time phishing detection system that classifies URLs as **SAFE** or **PHISHING** using Machine Learning and cybersecurity heuristics.

---

## Live Demo
- Frontend: https://phishguard-ai-three.vercel.app  
- Backend API: https://phishguard-ai-backend-62zz.onrender.com  
- ML Service: https://phishguard-ai-kuss.onrender.com/predict  

---

## Features
- Real-time URL scanning
- Phishing / Safe classification
- Confidence score visualization
- Dark cybersecurity UI
- Scan history stored in MongoDB
- Fully deployed architecture

---

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS
- **Backend**: Node.js, Express.js, MongoDB
- **ML**: Python, Flask, Scikit-learn
